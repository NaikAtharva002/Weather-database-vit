# Weather-database-vit
https://weather-atharva.streamlit.app/
Weather-database-vit is a Python-based dashboard app designed to visualize and analyze real-time weather data. Built with Streamlit, it offers an intuitive interface for users to view weather updates, explore data trends, and gain insights quickly. The app fetches data from public weather APIs and presents information using interactive plots and tables, making weather monitoring simple and accessible for students and enthusiasts.
